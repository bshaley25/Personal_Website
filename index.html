Math Maker

Math maker is an organizational tool for students with various disabilities to help write down and solve math problems. It uses a drag and drop style grid system with the ability to take pictures and save a current problem if needed. The biggest challenge for me was managing grid data and keeping the design simple. You can come check out the app here!

Math Maker Server Side

This was my first time implementing authentications with validations in Node.js using express. The third party libraries used are bcrypt and jwt to hash passwords and create tokens. Each user must have a unique username and the user features on the client side are only available given a valid token. This server features all CRUD functionality with knex SQL queries.

Asteriods!
